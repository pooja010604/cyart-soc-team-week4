# SOC Incident Response and Threat Hunting Lab

## Overview
This project demonstrates a complete Security Operations Center (SOC) setup using tools such as Wazuh, Elastic Security, MITRE Caldera, Metasploit, and TheHive. It covers both theoretical concepts and practical implementation of threat detection, incident response, and adversary simulation.

---

## Tools Used
- Wazuh (SIEM)
- Elastic Security (Kibana)
- MITRE Caldera
- Metasploit
- TheHive
- CrowdSec
- Velociraptor
- AlienVault OTX

---

## Key Activities

### 1. Threat Hunting
- Hypothesis-driven hunting (T1078)
- Log analysis using Elastic Security
- IOC validation using AlienVault OTX

### 2. SOAR Automation
- Playbook development for phishing response
- Automated IP blocking and case creation

### 3. Post-Incident Analysis
- Root Cause Analysis (5 Whys)
- Fishbone Diagram
- SOC metrics calculation (MTTD, MTTR)

### 4. Alert Triage
- Alert analysis in Wazuh
- Validation using VirusTotal

### 5. Evidence Analysis
- Network and process analysis using Velociraptor
- Chain-of-custody documentation

### 6. Adversary Emulation
- Attack simulation using MITRE Caldera
- Detection validation in Wazuh

### 7. Security Metrics and Reporting
- Dashboard creation using Elastic Security
- Executive-level reporting

### 8. Capstone Project
- Attack simulation using Metasploit
- Detection, triage, response, and reporting workflow

---

## Outcome
- Successful detection of simulated attacks
- Improved incident response workflow
- Practical understanding of SOC operations

---

## Conclusion
This project provides hands-on experience in threat detection, incident response, and SOC workflows, preparing for real-world cybersecurity roles.
